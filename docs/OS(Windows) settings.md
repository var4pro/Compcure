**RU:**
> Вместо настройки вручную безопасность системы обеспечивается с помощью **Microsoft Security Compliance Toolkit (MSCT)**.
> 
> - **Базовый уровень:** Базовый уровень корпоративной безопасности Windows 11, применяемый с помощью объекта групповой политики/LGPO.
>     
> - **Endpoint Protection:** Kaspersky Endpoint Security с политикой "По умолчанию запрещать" доступ к неизвестным исполняемым файлам.
